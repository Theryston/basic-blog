### 🔖 about

### 🚀 technologies used
- Node.js
- Javascript
- ejs
- MySQL

### 🗂️ installing

```bash

# cloning repository
git clone https://github.com/Theryston/blog.git

# accessing the project
cd blog

# installing dependencies
npm i

# now your need to create a database (using MySQL) with name blog.

# starting application
npm start
```
